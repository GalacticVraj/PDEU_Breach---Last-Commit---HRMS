# Setup Instructions

1.  **Install Dependencies**:
    Open this folder in VS Code, open a terminal, and run:
    ```bash
    npm install
    ```

2.  **Run the Portal**:
    ```bash
    npm run dev
    ```
    Access it at [http://localhost:3000](http://localhost:3000).

3.  **Credentials**:
    See `HANDOVER.md` for the full list of emails and passwords.

---
**Note**: This is a standalone copy. You can move this folder anywhere.
